# 战国风统一视觉设计手册

## 一、核心设计语言

### 1.1 统一视觉原则
- **材质**：竹简纹理、青铜质感、玉璧光泽、水墨晕染
- **色彩**：土黄(#D2B48C)为基底，七国专属色系，青铜(#CD7F32)点缀
- **纹样**：战国青铜器纹（云雷纹、夔龙纹、蟠螭纹）、竹简编绳、玉璧环纹
- **字体**：篆书为主，宋体为辅，数字用现代无衬线体
- **构图**：对称平衡，留白适当，战国器物比例

### 1.2 风格统一关键词
```
战国时期，竹简纹理，青铜质感，水墨晕染，战国纹样，对称构图，古朴典雅，历史厚重感，细节精致，4K分辨率
```

## 二、卡牌视觉系统

### 2.1 卡牌模板结构
```
卡牌尺寸：420×600像素（印刷标准）
边框：竹简样式，四角青铜包角，厚度8px
背景：羊皮纸纹理，轻微竹简纹路
插画区：300×300像素，居中
文字区：竹简色背景，黑色文字
属性区：圆形玉璧底，白色数字
```

| 稀有度 | 边框颜色 | 边框纹样 | 背景光效 | 特殊效果 |
| :--- | :--- | :--- | :--- | :--- |
| **普通** | 青铜色 #CD7F32 | 简单云雷纹 | 无 | 无 |
| **稀有** | 白银色 #C0C0C0 | 夔龙纹 | 微光 | 边框微光 |
| **史诗** | 黄金色 #FFD700 | 双龙戏珠 | 流光 | 动态流光 |
| **传说** | 玉色 #00FA9A | 龙凤呈祥 | 七彩光晕 | 动态光晕+粒子 |

## 五、统一生成工作流

### 5.1 批量生成策略
1. **先定基调**：生成基础材质图（竹简纹理、青铜质感、羊皮纸背景）
2. **统一模板**：制作卡牌模板、UI组件模板
3. **批量插画**：按职业批量生成卡牌插画
4. **组件生成**：生成图标、按钮、边框等UI组件
5. **特效动画**：最后生成动态特效

### 5.2 材质库建立
```
材质库/
├── textures/           # 基础材质
│   ├── bamboo_scroll.png      # 竹简纹理
│   ├── bronze_texture.png     # 青铜质感
│   ├── parchment.png          # 羊皮纸
│   └── jade_texture.png       # 玉石质感
├── patterns/           # 纹样库
│   ├── cloud_thunder.png      # 云雷纹
│   ├── kui_dragon.png         # 夔龙纹
│   ├── pan_chi.png            # 蟠螭纹
│   └── phoenix.png            # 凤凰纹
└── overlays/           # 叠加层
    ├── glow_normal.png        # 普通光效
    ├── glow_rare.png          # 稀有光效
    ├── glow_epic.png          # 史诗光效
    └── glow_legendary.png     # 传说光效
```

### 5.3 质量控制标准
1. **色彩一致性**：使用同一色板，偏差不超过±5%
2. **纹理统一**：竹简纹理方向、密度一致
3. **比例协调**：战国器物比例符合历史
4. **细节等级**：所有资源4K分辨率，边缘清晰
5. **文件格式**：PNG-24带透明通道，无损压缩

## 七、AI工具配置建议

### 7.1 Midjourney配置
```
--style raw          # 原始风格，减少AI过度美化
--ar 1:1/16:9/4:1   # 根据资源类型设置比例
--no modern, photo, 3D, cartoon, anime  # 排除现代元素
--chaos 20           # 适当随机性，避免完全一致
--seed 123456        # 固定种子，保证批次一致性
```

### 7.2 Stable Diffusion配置
```
Model: 国风模型（如GuoFeng3、Chinese-style-diffusion）
```
