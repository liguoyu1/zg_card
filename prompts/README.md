## 使用流程

1. **确定素材类型** → 英雄/随从/法术/武器/UI/场景/特效
2. **找到对应文件** → 参考上方索引
3. **搜索卡名** → 每个提示词前有卡ID（如 B001, F013）
4. **替换变量** → 如需微调，只修改[主体描述]部分
5. **添加通用标签** → 风格锚定 + 负面约束 (见各文件末尾)
6. **指定输出尺寸** → 参考输出规格表
7. **生成 + 验证** → 检查文字乱码/手部/构图/颜色

## 设计原则

1. **完全匹配游戏数据**: 每个提示词直接对应 `heroes_data.dart` 和 `cards/*.dart` 中的实际数据
2. **Flavor text 融入视觉**: 每个英雄/卡牌的 flavor text 转化为视觉描述
3. **职业差异化**: 每职业有独特的视觉语言（兵家=战场军事 / 法家=法庭律令 / 儒家=书院礼仪 / 道家=山水自然 / 墨家=工坊机械 / 阴阳家=星空宇宙 / 纵横家=外交谋略）
4. **无文字原则**: 所有提示词明确禁止中文文字，使用抽象符号代替
5. **风格一致性**: 所有卡牌共享核心风格锚定标签
