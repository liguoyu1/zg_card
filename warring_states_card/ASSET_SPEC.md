# 战国卡牌 — 美术/音效资产规格书

## 目录结构

```
assets/
├── heroes/       # 英雄头像 (128×128)
├── minions/      # 随从卡图 (自定义尺寸)
├── spells/       # 法术卡图 (自定义尺寸)
├── weapons/      # 武器卡图 (自定义尺寸)
├── gems/         # 稀有度宝石 (128×128 PNG)
│   ├── common.png      # 羊皮纸色 (八边形宝石)
│   ├── rare.png        # 翡翠绿 (八边形宝石)
│   ├── epic.png        # 铜褐 (八边形宝石)
│   └── legendary.png   # 金色 (八边形宝石+星芒)
├── icons/        # 关键词图标 (128×128 PNG)
│   ├── charge.png          # 闪电 ⚡ 黄
│   ├── taunt.png           # 盾牌 🛡 灰
│   ├── divine_shield.png   # 星芒 ✨ 白
│   ├── windfury.png        # 旋风 🌪 青
│   └── lifesteal.png       # 心形 ❤ 红
├── frame/        # 卡牌框纹理
│   └── card_frame.png      # 四角卷草纹+金边 (320×436)
├── back/         # 卡背图案
│   └── card_back.png       # 暗底金纹戰字封印 (80×98)
└── sounds/       # 音效 (Vorbis .ogg)
    ├── attack.ogg         # 攻击音效 (0.5s)
    ├── play_card.ogg      # 出牌音效 (0.3s)
    ├── damage.ogg         # 受伤音效 (0.3s)
    ├── death.ogg          # 死亡音效 (0.5s)
    ├── heal.ogg           # 治疗音效 (0.4s)
    ├── end_turn.ogg       # 回合结束 (0.3s)
    ├── victory.ogg        # 胜利 (2s)
    ├── defeat.ogg         # 失败 (2s)
    ├── mana_crystal.ogg   # 法力水晶 (0.3s)
    ├── button_click.ogg   # 按钮点击 (0.15s)
    ├── card_slide.ogg     # 卡牌滑动 (0.2s)
    └── coin.ogg           # 硬币掉落 (0.3s)
```

## 1. 🎨 稀有度宝石 (gems/)

### 尺寸
- **源文件**: 128×128 px (SVG)
- **在游戏中显示**: 14×14 px (Card左上角叠加)

### 颜色规范
| 稀有度 | 主色 | 高光色 | 边框色 |
|--------|------|--------|--------|
| Common (普通) | `#C4B085` | `#E8D5B7` | `#A89060` |
| Rare (稀有) | `#4A7C59` | `#6BAF7A` | `#2D5C3F` |
| Epic (史诗) | `#8B4513` | `#C7853C` | `#6B3410` |
| Legendary (传说) | `#C59538` | `#F0D060` | `#A07620` |

### 形状规范
所有宝石用**八边形**（八角形），上下各90°角，左右各45°角。
传说级加两个 ✦ 星芒标记在顶部。

### 高光效果
- 上半部渐变高光 (60%透明度到0%)
- 中心菱形反光 (30%白)

## 2. 🎯 关键词图标 (icons/)

### 尺寸
- **源文件**: 128×128 px (SVG)
- **在游戏中显示**: 12×12 px (Card左上角叠加)

### 图标定义
| 关键词 | 图形 | 主色 | 风格 |
|--------|------|------|------|
| 冲锋 (charge) | 闪电 | `#FFD700` 金黄 | 白发光描边 |
| 嘲讽 (taunt) | 盾牌 | `#BDBDBD` 灰 | 盾面六角纹 |
| 圣盾 (divineShield) | 五角星 | `#FFFFFF` 白 | 金色描边+发光 |
| 风怒 (windfury) | 旋风 | `#80DEEA` 青 | 涡旋造型 |
| 吸血 (lifesteal) | 心形 | `#EF5350` 红 | 深红描边+高光 |

## 3. 📐 卡牌框 (frame/)

### 尺寸
- **源文件**: 320×436 px (可缩放到任何卡牌尺寸)
- **游戏中使用**: `BoxFit.fill` 拉伸铺满

### 设计规范
- **底色**: 半透明羊皮纸渐变 (覆盖在卡图上不遮挡)
- **外框线条**: 渐变金边 (3px宽)
- **四角装饰**: 金线卷草纹 + 中心小金点
- **上下装饰线**: 顶部/底部细金线

## 4. 🃏 卡背 (back/)

### 尺寸
- **源文件**: 80×98 px (标准卡尺寸)
- **游戏中作为**: 对手手牌数量徽章背景

### 设计规范
- **底色**: `#3D2B1F` 深褐色, 左下到右上渐变
- **外框**: 金边 (1.5px)
- **中心图案**: 圆形封印, 内刻"戰"字 (金色)
- **四角落**: 45°旋转菱形装饰

## 5. 🔊 音效 (sounds/)

### 技术规范
- **格式**: Ogg Vorbis (.ogg)
- **采样率**: 44100 Hz
- **声道**: 单声道 (Mono)
- **编码质量**: Q5 (标准质量)

### 音效简表
| 文件名 | 时长 | 描述 | 风格 |
|--------|------|------|------|
| `attack.ogg` | ~0.5s | 刀剑相击 | 金属碰撞 |
| `play_card.ogg` | ~0.3s | 纸牌飞出 | "刷"一声 |
| `damage.ogg` | ~0.3s | 受击 | 闷响 |
| `death.ogg` | ~0.5s | 消散 | 下降音调 |
| `heal.ogg` | ~0.4s | 治疗 | 上升音调 |
| `end_turn.ogg` | ~0.3s | 回合切换 | 钟/铃 |
| `victory.ogg` | ~2s | 胜利 | 号角/凯旋 |
| `defeat.ogg` | ~2s | 失败 | 低沉 |
| `mana_crystal.ogg` | ~0.3s | 法力成长 | 水晶音 |
| `button_click.ogg` | ~0.15s | 按钮 | 清脆点击 |
| `card_slide.ogg` | ~0.2s | 卡牌滑动 | 滑音 |
| `coin.ogg` | ~0.3s | 硬币 | 金属叮声 |

### BGM 需求 (尚未实现)
- **格式**: Ogg Vorbis, 单声道, 44100 Hz
- **时长**: 30-60秒 loop
- **风格**: 战国/中国古代风, 用古筝/笛/鼓等乐器
- **音量**: 游戏内 0.3 (小声垫底)
- **文件命名**: `bgm.ogg` → 放入 `assets/sounds/`
