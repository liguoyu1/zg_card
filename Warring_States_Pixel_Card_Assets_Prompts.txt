## ===== Z-IMAGE-TURBO PARAMETERS =====
# Recommended settings for pixel art:
# - num_inference_steps: 8 (balanced quality/speed)
# - acceleration: high (for fast generation)
# - image_size: portrait_4_3 or square
# - No negative prompts (Z-Image-Turbo doesn't support them)
# - All constraints must be in positive prompt

## ===== PRODUCTION WORKFLOW =====
1. Generate character sprites (120×160px, transparent BG)
2. Generate card borders (195×284px, transparent center)
3. Generate weapon icons (64×64px, transparent BG)
4. Generate stat icons (32×32px, transparent BG)
5. Generate card backs (195×284px, full design)
6. In game engine: Layer character + border + UI elements
7. Add text programmatically (English card names/effects)
