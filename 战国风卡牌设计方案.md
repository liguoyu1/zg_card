# 战国风卡牌设计方案

## 一、卡牌设计原则

### 1.1 设计理念
- **历史真实性**：卡牌名称、效果、插画均基于战国历史典故
- **游戏平衡性**：遵循炉石传说式费用-强度曲线
- **文化融合**：将战国元素与卡牌游戏机制有机结合
- **视觉统一**：保持战国风格的美术一致性

### 1.2 卡牌类型分布
| 类型 | 数量 | 占比 | 说明 |
| :--- | :--- | :--- | :--- |
| 随从 | 100张 | 50% | 兵种、名臣、名人、神兽 |
| 法术 | 60张 | 30% | 计策、阵法、天象、仪式 |
| 武器 | 20张 | 10% | 战国兵器、礼器、法器 |
| 英雄 | 20张 | 10% | 七国王侯、百家宗师 |
| **总计** | **200张** | **100%** | 基础卡池 |

### 1.3 稀有度分布
| 稀有度 | 数量 | 占比 | 颜色 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| 普通 | 100张 | 50% | 青铜色 | 基础兵种、常见法术 |
| 稀有 | 60张 | 30% | 白银色 | 精锐部队、高级计策 |
| 史诗 | 30张 | 15% | 黄金色 | 名将、重要事件 |
| 传说 | 10张 | 5% | 玉色 | 君主、宗师、神迹 |
